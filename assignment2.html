Bella's A1: Laser Cut Press Fit Construction Kit!

Process

  • Step 1: Concept hand drawing

    • This was helpful to see the visualization on paper before starting.

  • Step 2: Fusion360 + Illustration File

    • I used Fusion360 to draw in real size and converted the file to dxg and opened in Illustration to change the line weight and color.

  • Step 3: Laser Cut and Adjustment

    • I laser cutted a few pieces and learned that I need to change the model to make sure it fits better. I went back to Step 2 and Step 3 two-three times.

Source File

Illustrator file Fusion360 file

Machine Settings: Material Soft Wood, Thickness: 006"

Kit Design and Execution

  • Design Description: My design idea came from the shape of post-its twirl. I wanted to mimic this shape.
  • Execution: I sketched out the drawing and knew I need to make 2D rectangles to make 3D rectangles and stack them against each other. It was hard to put it together because I did not take into consideration of different angles.
